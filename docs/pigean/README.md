# PIGEAN Docs Namespace

This directory is the target home for PIGEAN-specific documentation as the repo is reorganized around:

- `docs/pigean/`
- `docs/eaggl/`

During transition, existing top-level docs such as:

- `docs/ADVANCED_SET_B.md`
- `docs/EAGGL_INTEROP.md`
- `docs/CLI_OPTIONS.md`

remain authoritative until they are moved or regenerated in the namespaced layout.
